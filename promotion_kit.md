>
> I developed an open-source meta-skill for AI agent engines (such as Claude Code, Antigravity, and Codex CLI) that automates the lifecycle of creating custom agent skills.
> 
> Creating a custom skill typically requires repetitive manual work: writing skill definitions, compiling cross-platform installation scripts, translating readmes, and pushing files to remote repositories.

> 1. **Design & Scripts**: Generates the custom `SKILL.md` and programmatic helper scripts based on a natural language prompt.
> 2. **Installer Bundling**: Generates native compatible shell (`install.sh`) and PowerShell (`install.ps1`) installers.
> 3. **5-Language Docs**: Generates interlinked READMEs in English, Korean, Japanese, Spanish, and Chinese with Shields.io badges.
> 4. **Git Sync**: Initializes a local Git repo and pushes it to your remote GitHub path.
